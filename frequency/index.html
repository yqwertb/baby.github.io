# 1 {}、new Object()，Object.create({})的区别

  • {}

    字面量创建对象和new Object()一样，会将创建对象的__proto__链接到Object()的原型对象prototype上

  • new Object()

    new的过程

    1. 创建一个空对象
    2. 将构造函数(Object())的原型链接到空对象的__proto__上
    3. 绑定构造函数的this到新对象上
    4. 返回对象

  • Object.create(obj, preproties)

    将创建的对象的__proto__链接到obj的原型对象prototype上。是浅拷贝

new Object()、Object.create()、{}三种对象创建方式的区别 (opens new window)

# 9 css的position属性

分为这几个值：static、relative、absolute、fixed、sticky

  • static 没有定位，元素处于正常文档流中
  • relative 相对定位，相对自己的初始位置的左上角
  • absolute 绝对定位，相对于离自己最近的非static定位祖先元素
  • fixed 固定定位，脱离文档流，相对于浏览器定位
  • sticky 吸顶，元素根据正常文档流定位，然后根据最近的滚动祖先，基于top,bottom,left,right`进行偏移

top,bottom,left,right百分比相对谁

top、bottom相对于高度，left、right相对于宽度

relative和absolute 相对于父元素的宽高

fixed相对于最近的滚动祖先的宽高

# 13 H5和小程序的区别

  • 运行环境：H5是在浏览器中运行，小程序在软件中运行
  • 系统权限：小程序借助于宿主程序，实现的权限更多更全
  • 开发成本：小程序集成了一些api，并且不用考虑浏览器的兼容性问题
  • 渲染：
    • 小程序是双线程，一个线程专门处理视图，一个执行js
    • 浏览器是单线程，js的执行会阻塞页面的渲染

# 14 SSR的优缺点

  • 首屏加载时间短，服务器拼接好html字符串，浏览器只需要渲染
  • 利于seo
  • 过于占用服务器资源
  • 不利于前后端分类

# 16 水平垂直布局

# 已知元素宽高

  1. 父元素绝对相对定位 ，子元素绝对定位，子元素定位距顶距左50%，margin偏移顶部左部的值是宽高的-50%

    top，left的值是相对于父元素宽高的

    margin设置负值

  2. 父元素相对定位 ，子元素绝对定位，子元素top、bottom、right、left设置0，margin设置auto

# 未知元素宽高

  1. 父元素相对定位，子元素绝对定位，子元素设置定位距顶距左50%，设置transform: translate(-50%, -50%)

# 17 margin塌陷和合并

# 塌陷

两个div嵌套，外层div没有文字，没有padding-top，没有border，则会发生margin的塌陷，如果同正同负，取绝对值最大的值作用于外层div。如果一正一负，是两个的计算值作用于外层div。

# 合并

两个相邻的div，上方的margin-bottom和下方的margin-top，如果同正同负，取绝对值最大的值。如果一正一负，是两个的计算值。
