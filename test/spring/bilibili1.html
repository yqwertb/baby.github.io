                            # 区别

                            浅拷贝：基本类型是值，引用类型是内存地址；深拷贝：开辟新的内存，和原变量两个内存

                            # 实现

                            浅拷贝：Object.assign；展开运算符 深拷贝：JSON.parse(JSON.stringify())；递归
